# 이미지 유사도 검사 프로그램

C# WinForms와 OpenCVSharp를 활용한 이미지 유사도 검사 프로그램입니다.

사용자가 검사 이미지와 템플릿 이미지를 불러온 후 ROI를 지정하면

해당 영역을 비교하여 유사도를 계산하고 PASS/FAIL을 판정합니다.

검사 결과를 CSV 로그 파일로 저장하고,FAIL 발생시 검사 이미지를 자동 저장합니다.

# 주요 기능

CCoeffNormed 기반 이미지 유사도 검사(ROI,템플릿)

ROI 영역 지정 및 저장,불러오기 기능

유사도 기준값 설정 및 PASS/FAIL 판정

FAIL 당시 이미지(템플릿,ROI) 자동 저장 기능

검사 이력 관리 기능
